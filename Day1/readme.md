On Day 1 of your DSA journey in C++, you learned the fundamentals of programming that form the base of all Data Structures and Algorithms. You understood the structure of a C++ program, how main() works, and how to use #include <iostream> with cout and cin for input and output. You practiced variables, data types like int, float, char, and string, along with arithmetic operators such as +, -, *, /, and %. You also learned decision-making using if-else statements and repetition using loops like for and while. After learning these basics, you solved beginner problems such as checking even or odd numbers, printing numbers from 1 to N, finding the sum of first N numbers, checking positive or negative numbers, and printing multiplication tables. Day 1 mainly focused on building basic coding confidence and understanding program logic.
