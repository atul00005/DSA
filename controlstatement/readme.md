On Day 2, you moved into pattern printing, which is one of the best ways to improve logical thinking and nested loop understanding in DSA. You learned how nested loops work, where the outer loop controls rows and the inner loop controls columns. Using this logic, you created different patterns such as square star patterns, triangle patterns, reverse triangles, number patterns, pyramids, inverted pyramids, diamonds, Floyd’s triangle, hollow squares, and the butterfly pattern. You also learned how to analyze patterns by observing how rows, spaces, and stars change in each iteration. Day 2 strengthened your thinking ability, loop control, and problem visualization skills, which are extremely important for advanced DSA topics later like arrays, recursion, trees, and dynamic programming.
